# Dict2Anki
**只支持 Anki 2.0.x.**  
**Dict2Anki** 是一款方便[有道词典](http://cidian.youdao.com/multi.html)、[欧陆词典](https://www.eudic.net/)用户同步生成本地单词本卡片至[Anki](https://apps.ankiweb.net/#download)的插件  

## 特点介绍
- 支持导入有道词典、欧陆词典生词本
- 支持Mac、Windows平台
- 检测词典软件生词变化在Anki中相应的添加删除卡片

## 如何安装
- ### 从源码安装（推荐）
    - 在[Release]()中下载最新版本
    - 解压`Zip`
    - 把`Dict2Anki.py` 放入Anki的`Addons`目录
    - 重启Anki
- ### 从AnkiWeb安装
    - 打开 `工具` -> `插件` -> `浏览&安装` 
    - `代码` : `480644339`
    - 重启Anki
